What is tyson ?
---------------

`tyson` is a tool that converts ASCII diagrams into Unicode diagrams using box
drawing characters.

For example you can feed the following into `tyson` :

    +---+      +---+
    | A | ---> | B |
    +---+      +---+

and get the following result :

    ┌───┐      ┌───┐
    │ A │ ╶──▷ │ B │
    └───┘      └───┘
